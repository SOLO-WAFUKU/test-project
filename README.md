# Test Project

このプロジェクトは、基本的なJavaScriptプログラムの学習を目的としたサンプルコードを含んでいます。

## プログラムの説明

### index.js
1から10までの数値を合計するプログラムです。forループを使用して数値を累積し、結果をコンソールに出力します。

**実行方法:**
```bash
npm start
```
または
```bash
node src/index.js
```

**出力結果:**
```
1から10までの合計: 55
```
